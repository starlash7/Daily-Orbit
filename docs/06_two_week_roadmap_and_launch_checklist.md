## 2) QA 체크리스트

- [ ] 무료 1회 제한 정확히 동작
- [ ] 카테고리별 결과 정상 생성
- [ ] 공유 버튼/링크 정상 동작
- [ ] streak 증가/초기화 규칙 정상 동작
- [ ] 유료 심화 리딩 잠금/해제 정상 동작

## 예외 QA

- [ ] 네트워크 끊김 시 재시도 가능
- [ ] 결제 취소 시 자연스러운 복귀
- [ ] 결제 실패 메시지 명확
- [ ] 중복 요청 방지

## 모바일 QA

- [ ] 로딩 3초 이내(핵심 화면 기준)
- [ ] 터치 타겟 44px 이상
- [ ] iOS/Android 주요 브라우저 호환

- [ ] account association 검증 완료

- [ ] 공유 미리보기 정상 노출
- [ ] sponsored tx/paymaster 적용 여부 점검
- [ ] 결제 성공/실패 케이스 리허설 완료

## 4) Go/No-Go 기준

Go 조건:

- P0 버그 0건
- 결제 성공률 기준 충족(내부 테스트)
- 이벤트 수집 정상
- 핵심 퍼널 측정 가능

No-Go 조건:

- 결제 실패 처리 미완성
- 무료 뽑기 제한 오류
- 공유 기능 주요 기기에서 불가

## 5) 런칭 직후 72시간 운영

1. 장애/이탈 모니터링 (실시간)
2. 퍼널 수치 확인 (6시간 간격)
3. 결제 오류 로그 즉시 대응
4. 카피/가격 작은 실험 1개만 진행

## 6) 가정

- 디자인/개발 리소스가 병렬 투입 가능
- 배포 권한/도메인/결제 계정 사전 준비 완료
- 런칭 시점은 기능 안정성 우선
