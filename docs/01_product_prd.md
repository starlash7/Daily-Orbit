# 01. Product PRD

## 1) 제품 정의

- 제품명: Daily Orbit
- 제품 유형: Base Mini App 기반 일일 운세 앱
- 핵심 목표: "무료 체험 -> 공유 -> 재방문" 루프 검증

## 2) 문제 정의

| 문제 | 현재 불편 | 해결 방향 |
|---|---|---|
| 재방문 동기 부족 | 운세 앱은 1회성 사용으로 끝나는 경우가 많음 | 일일 무료 + streak + 리마인드 루프 |
| 공유 전환 낮음 | 텍스트 중심 결과는 공유 매력이 약함 | 카드형 결과 이미지 생성/공유 |
| 결제 거부감 | 초반부터 과한 paywall은 이탈 유발 | 코어 무료, 심화만 소액 유료 |

## 3) 목표/비목표

### 목표 (MVP)

- 하루 1회 무료 운세 뽑기 경험을 10초 이내로 제공
- 결과 공유율을 의미 있는 수준으로 확보
- 심화 리딩 유료 전환 가설 검증

### 비목표 (MVP)

- 정교한 점성학 엔진 구축
- 대규모 소셜 네트워크 기능
- 멀티 캐릭터/복잡한 아바타 시스템

## 4) 타깃 사용자

1. 짧은 콘텐츠를 빠르게 소비하는 모바일 사용자
2. "재미 + 감성" 기반으로 결과를 공유하는 사용자
3. 매일 루틴형 앱(출석/퀴즈/운세)을 사용하는 사용자

## 5) 사용자 가치 제안

- 1분 안에 오늘의 분위기와 행동 힌트를 얻는다
- 결과가 보기 좋고 공유하기 쉽다
- 부담 없는 소액 결제로 더 깊은 해석을 받을 수 있다

## 6) 핵심 루프

```text
앱 진입 -> 무료 운세 뽑기 -> 결과 확인 -> 공유
   -> (선택) 심화 리딩 구매 -> streak 적립 -> 다음날 재진입
```

## 7) 기능 범위

### MVP (필수)

| 기능 | 설명 | 우선순위 |
|---|---|---|
| 무료 일일 뽑기 | 사용자당 1일 1회 무료 운세 | P0 |
| 카테고리 4종 | 연애/금전/커리어/건강 | P0 |
| 결과 카드 | 한줄 운세 + 행운 요소 + 캐릭터 | P0 |
| 공유 | 링크/이미지 공유 | P0 |
| streak | 연속 방문/뽑기 일수 집계 | P0 |
| 심화 리딩 유료 | 상세 해석 잠금 해제 | P1 |

### v1.1

- 운세 히스토리
- 주간 요약 카드
- 공유 템플릿 추가

### v2+

- 친구 운세 비교
- 궁합 모드
- 캐릭터 커스터마이징

## 8) Out of Scope

- NFT 발행/거래
- 토큰 자체 발행
- 오프체인 광고 네트워크 연동

## 9) 콘텐츠 정책 가이드

- 단정적 예언/공포 유도 문구 금지
- 건강/금융 관련 표현은 조언 중심으로 완화
- 혐오/차별/폭력 표현 금지

## 10) 가정

- 결제는 USDC 기준으로 운영
- 런칭 초기에는 한국어 단일 언어
- 마케팅은 공유 카드 중심의 유기적 유입 우선

## 11) 리스크 및 대응

| 리스크 | 영향 | 대응 |
|---|---|---|
| 리텐션 저조 | 루프 붕괴 | streak 보상 강화, 리마인드 메시지 A/B |
| 유료 전환 저조 | 수익성 약화 | 가격 구간 테스트(0.3/0.5/0.7), paywall 카피 개선 |
| 콘텐츠 반복 체감 | 만족도 하락 | 카테고리별 문구 풀 확장, 주간/시즌 테마 |
